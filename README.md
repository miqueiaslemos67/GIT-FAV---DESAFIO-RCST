## About

GitFav is a web application developed as part of a Rocketseat challenge, designed to help users manage and track their favorite GitHub profiles. This project aims to provide users with an intuitive interface to add, view, and remove favorite profiles, fetching real-time data directly from GitHub's API.

## Features

- **Add GitHub Profiles:** Easily add profiles by entering a username.
- **Real-time Data Fetching:** Instantly view profile information such as repositories, followers, and following counts.
- **Profile Management:** Quickly manage your favorites, with options to view detailed profile information or remove profiles from your list.
- **Responsive Design:** Optimized for use across various devices, ensuring a seamless experience on desktop, tablet, and mobile.

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub API

## Getting Started

### Installation

Clone this repository:
```bash
git clone https://github.com/yourusername/GIT-FAV---DESAFIO-RCST.git
```

Open the `index.html` file in your browser to start using the application.

### Usage

- Input a GitHub username in the provided field and click the "Add" button.
- View the profile's data immediately updated on the list.
- Manage profiles by removing them or exploring more details directly via GitHub.
